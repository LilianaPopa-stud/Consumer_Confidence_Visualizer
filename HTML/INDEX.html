Visualize Consumer Confidence Data with CoCo

CoCo is a flexible web-based tool that allows you to easily visualize consumer confidence data using our REST/GraphQL API. Compare data across states and additional indicators to create beautiful visualizations that can be exported in CSV, WebP, and SVG formats.

Features

  • Flexible Visualization: Choose from multiple visualization types.
  • Data Comparison: Compare data across different countries.
  • Export Functionality: Export your visualizations in CSV, WebP, and SVG formats for easy sharing and collaboration.
